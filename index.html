Improve GDB's usability with PPP Debugger
Plush Puppy Parallel Debugger will provide a full environment to your debugging session.
  • Debug parallel codes from C language using pthread or OpenMP libs or just regular C programs
  • Easy change of threads context with intuitive buttons
  • Inspect expressions value from different threads
  • Command execution flow controlling all threads as GDB's default or just the selected threads for better debugging
  • See advanced information such as the assembly code for the current thread or the list of values of the process registers
